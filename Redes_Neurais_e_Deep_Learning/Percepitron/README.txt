Autor: Vitor de Almeida Silva


Implementação de um neurônio ou perceptron para treinamento. O programa segue a lógica do perceptron comparando os resultados com o erro e atualizando os pesos a cada iteração. A base de dados consiste de uma matriz de característica que é inserida diretamente no código.

O código pode ser alterado para treinamento com diferentes Matrizes de características e resolucação de diferentes problemas.
