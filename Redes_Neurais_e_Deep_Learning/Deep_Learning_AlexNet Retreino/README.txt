Autor: Vitor de Almeida Silva

Código para retreino por transfering learning da rede neural AlexNet. Este programa está programado para ser treinado com uma pequena base de dados de duas classes contida na pasta images. O programa pode ser alterado para ser treinado com qualquer base de dados de N classes, também podem ser configurados a porcentagem de imagens para treino e a porcentagem de imagens para teste.

Para mais informações leia o relatório na da pasta anterior
